# mini-proyects

Backend projects to learn production design patterns and microservices in practice.

Each project focuses on one or two patterns — small, local, dependency-free.

## Patterns

- Adapter
- Strategy
- State machine
- Circuit breaker + bulkhead
- Outbox + idempotency
- Saga
- CQRS (lightweight)
- Decorator / middleware

## Setup

- Node.js + TypeScript
- SQLite
- Docker Compose (for microservices)
- Zero paid services
